# Frontend

Frontend da aplicação desenvolvido com Next.js e TypeScript.

---

# 🚀 Tecnologias

- Next.js
- React
- TypeScript
- Bun
- Biome

---

# 📁 Estrutura

```txt
frontend/
├── src/
│   ├── app/
│   ├── components/
│   ├── lib/
│   └── styles/
│
├── public/
├── package.json
└── README.md
```

---

# ⚙️ Instalação

Instale as dependências:

```bash
bun install
```

---

# 🧪 Rodando em desenvolvimento

```bash
bun run dev
```

---

# 🌐 Aplicação

```txt
http://localhost:3000
```

---

# 🏗️ Build

```bash
bun run build
```

---

# ▶️ Produção

```bash
bun run start
```

---

# 🧹 Formatação

## Verificar código

bun run lint

bun run format
